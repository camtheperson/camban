## Requirements

* [Docker Desktop](https://www.docker.com/products/docker-desktop/)

## Getting Started

First, build the development Docker image.

```bash
make build-dev
```

Then, run it:

```bash
make run-dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
